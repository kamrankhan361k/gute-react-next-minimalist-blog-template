How to install

What you need
  • NodeJs 14.16.1 or higher
Install step
  1. In Gute's root folder install all pakages with command npm install
  2. Run local RestAPI server with command npm run server
  3. Run your poject in development mode with command npm run dev
Build & Test
  • To build app to check before you deploy to your host you just need run npm run build
